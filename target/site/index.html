About Multiagent Programming - Group 3 - SS 22 FernUniversität in Hagen

This site provides a technical documentation of the project "Multiagent Programming - Group 3" which was created by four computer science students at the Fernuniversität in Hagen Summer 2022. It contains data about project dependencies and code documentation in the form of JavaDoc, which can be found in the "Project Reports" section.
